# wechat-autopicture
微信图片搜索，自动回复图片

关键字 `图 xxx`，比如要搜索`篮球`，回复：`图 篮球`，搜索新恒结衣，回复：`图 新恒结衣`

微信接口基于 [wechat4u](https://github.com/nodeWechat/wechat4u)

## Usage

- npm install
- node index
